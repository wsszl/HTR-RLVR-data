Data used in the reinforcement learning phase for difficult maps
Each map's folder contains two subfolders: `prompt` and `rlRewardActionData`. The `prompt` folder contains data in dictionary format, where the key is the sequence number and the value is the prompt from the large model. The `rlRewardActionData` folder also contains data in dictionary format, where the key is the sequence number and the value is the optimal action sequence given by the reinforcement learning forward propagation. The files in both folders are aligned using the sequence numbers, linking the prompt and action sequences, and used as a scoring reference for the large model's output.
